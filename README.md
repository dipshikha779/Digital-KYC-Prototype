## Problem Statement
High user drop-off during Digital KYC due to document scan failures, upload issues, and verification delays.



## Solution Overview
This project presents a mobile-first guided Digital KYC onboarding prototype designed to:
- Reduce document scan errors
- Prevent upload failures
- Improve trust during verification



## Key Features
- Guided document scanning with alignment & lighting feedback  
- AI-enhanced document preview before upload  
- Upload with progress bar and retry attempts  
- Selfie verification with blink detection  
- Real-time KYC verification status with ETA  
- Final success confirmation screen



## Target Users
Youth and first-time digital banking users.



## Tools & Technologies Used
- Figma (UI/UX & Live Interactive Prototype)
- React + TypeScript (UI Flow Simulation)
- AI-assisted UI generation (design acceleration only)
